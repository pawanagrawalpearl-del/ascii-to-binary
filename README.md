# 📟 Binary Translator

A sleek, minimalist web utility that converts text, numbers, and sentences into 8-bit binary code in real-time.

[Live Demo](https://yourusername.github.io/your-repo-name/)

## ✨ Features
* **Real-time Conversion:** See results as you type.
* **Modern UI:** Responsive design with a dark mode aesthetic.
* **One-Click Copy:** Easily copy binary strings to your clipboard.
* **Character Counter:** Keep track of your input length.

## 🛠️ Built With
* HTML5
* CSS3 (Flexbox & Custom Properties)
* Vanilla JavaScript

## 🚀 How to Use
1. Enter your text in the input box.
2. The binary code will generate automatically.
3. Click "Copy Result" to save it to your clipboard.
